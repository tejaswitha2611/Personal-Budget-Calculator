# 💰 Personal Budget Calculator

A desktop application built with Python to track income, expenses, savings, and investments.

## Features
- Track income and expenses
- Set budget limits per category
- Daily / Weekly / Monthly expense summaries
- Savings calculation and goal tracker
- Investment recommendations (SIP, FD, PPF, Gold)

## Tech Stack
- Python
- Tkinter (GUI)
- SQLite (Database)
- Matplotlib (Charts)

## How to Run
1. Clone the repository
2. Install dependencies: pip install matplotlib tkcalendar pillow
3. Run: python main.py
